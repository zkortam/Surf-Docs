# Post Components

When you long-press a post, the **Post-Control Widget** pops up. Within it, you can do the following interactions:
1. Share the post. If you're on mobile, this will open your sharing menu.
2. Copy the link to the post.
3. If it's your post, you'll be able to edit the caption.
4. You can save the post to your library.
